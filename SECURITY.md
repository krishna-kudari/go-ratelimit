# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.**

Email: **krishna.kudari@outlook.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours. Critical issues will be patched and released as soon as possible.

## Supported Versions

| Version | Supported |
|---------|-----------|
| v1.x    | Yes       |
| < v1.0  | No        |
